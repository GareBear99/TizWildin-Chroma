# TizWildin Chroma Loops&Sample Packs

**TizWildin Chroma** is a multi-segment synthwave loop sample pack from **TizWildin Entertainment / Gary Doman**.

It consolidates the full Chroma line into one flagship repo-style release covering multiple neon lanes instead of a single narrow genre slice.

## What Chroma is

- A curated loop-first synthwave sample pack line
- Built as a segmented color/mood system rather than one flat folder of random loops
- Designed for producers, editors, trailer makers, game-audio builders, and sample-based composers
- Structured for drag-and-drop use, stem extraction, and fast sketching inside a DAW

## Core concept

Chroma is organized as **multiple tonal lanes**:

1. Classic driving synthwave
2. Atmospheric VHS dreamside
3. Crisp clean premium neon
4. Dark aggressive chase/combat
5. Emotional romantic cinematic neon
6. Dark eerie godlike ritual
7. Dark eerie godlike violin-fused
8. Violin-first legato correction / lament lane

Together these create a wider palette than a normal synthwave pack:
- outrun and night-drive
- cinematic ambience
- combat/boss energy
- emotional neon scoring
- ritual dread
- violin-led dark beauty

- synthwave and retrowave production

## Technical notes

- Format: WAV
- Channels: stereo
- Bit depth: 16-bit PCM
- Sample rates vary by segment/source pack build
- Includes loops, alternate mixes, and selected stem-style extras
